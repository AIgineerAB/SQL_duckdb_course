# SQL course

In this course, you'll dive into fundamental SQL including

- querying
- filtering
- aggregations
- joins
- CTEs
- connecting to Python

For this you will be using the following tools:

- duckdb
- dbeaver
- duckdb CLI
- python
- bash
- git
- github
